# Automação de Testes Web com Cypress

Este repositório documenta minha evolução prática em **automação de testes web com Cypress**, com base no curso **Automação Web com Cypress da Qazando Academy**.

O objetivo do projeto é consolidar conhecimentos em **testes automatizados end-to-end (E2E)**, organização de cenários de testes, boas práticas de automação e estruturação de testes utilizando **JavaScript e Cypress**.

## Objetivos do projeto

* Praticar automação de testes web (E2E)
* Aplicar boas práticas na escrita de testes automatizados
* Organizar cenários de testes de forma clara e escalável
* Evoluir continuamente adicionando novos cenários e melhorias
* Utilizar Git e GitHub para versionamento de código

## Tecnologias e ferramentas

* JavaScript
* Cypress
* Node.js
* FakerJS
* BDD com Cucumber
* Git
* GitHub

## Conceitos praticados

Durante o desenvolvimento deste projeto são praticados conceitos importantes de automação de testes, como:

* Escrita de cenários de testes E2E
* Assertions (validações)
* Inspeção de elementos
* Gestão de dados de testes com **fixtures**
* Geração de dados dinâmicos com **FakerJS**
* Uso de **hooks (beforeEach)**
* Estruturação de testes com **Page Objects**
* Criação de **comandos personalizados**
* Escrita de testes em **BDD com Gherkin**
* Organização de steps com **Cucumber**
* Simulação de dispositivos mobile
* Geração de **relatórios de execução**
* Integração com **Git**
* Pipelines CI/CD (GitHub Actions, notificações no Slack e agendamento de execução)


## Como executar o projeto

### 1. Clonar o repositório

```
git clone <url-do-repositorio>
```

### 2. Instalar as dependências

```
npm install
```

### 3. Executar o Cypress

```
npx cypress open
```

## Evolução do projeto

Este repositório não se limita apenas aos exercícios do curso.

Com o avanço dos estudos, novos cenários de testes, melhorias na estrutura do projeto e aplicação de boas práticas de automação serão adicionados continuamente.

O objetivo é utilizar este projeto como ambiente de prática para aprofundar conhecimentos em **automação de testes com Cypress**, explorando novas abordagens, refatorações e cenários além dos propostos inicialmente no curso.
